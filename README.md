# OPN server generator
## Usage
```
$ go build
$ ./opn-generator new [projectName] --database [/path/to/plantUMLFile] --api [/path/to/apiSpecYAMLFile]
```

where `--database`, `--api` are options.

New project will be created in the same directory.
